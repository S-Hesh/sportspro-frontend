## Features
- Profile, posts/feed
- Networking: connect, accept, remove (Friends)
- Opportunities

## Run locally
1. Copy `.env.example` → `.env` and set `VITE_API_BASE_URL=http://localhost:8080/api`
2. `npm install`
3. `npm run dev`
